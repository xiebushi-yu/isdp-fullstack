# 项目名称：便利店 POS 系统 - 前端

## 项目简介

这是一个基于 Vue 3 + TypeScript 的前端课程设计，实现了用户登录、商品管理、销售管理等核心功能。

## 技术栈

- Vue 3
- TypeScript
- Pinia
- Vue Router
- Vite
- Element Plus 

## 功能模块

- 用户登录/注册
- 商品列表展示
- 商品查询
- 销售

## 如何本地运行

1. 克隆项目

   ```bash
   git clone https://github.com/xiebushi-yu/isdp-fullstack.git

   ```

2. 前端启动
   ```bash
   cd isdp-vue3

   ```

3. 安装依赖
   ```bash
   pnpm install
   ```

4. 启动开发服务器
   ```bash
   pnpm dev
   ```
